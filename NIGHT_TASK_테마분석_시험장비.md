## Task A. 업무 키워드 → 테마 매핑

### 목적
프로젝트별이 아닌 **역량 테마별**로 포트폴리오를 재구성하기 위한 기초 작업.
서로 다른 프로젝트(SCU, EOP, SS500 등)에서 같은 결의 업무를 묶어 "이 사람은 이걸 반복적으로 잘한다"를 보여준다.

### 추출된 프로젝트별 업무 키워드 (확인 완료)

| 프로젝트 | 핵심 업무 키워드 |
|----------|-----------------|
| **SCU** (현대/DIC) | PCB 설계(ES→AS→A2S 3세대), BLDC 구동, CAN DB(CANoe/BUSMASTER), EMC/환경시험(ES95400/ES96200), Infineon TC23x, SBW 부하시험 |
| **EOP 400W** (국책과제) | SVPWM 절환, 온도별 성능시험(55°/85°/120°), CAN Sleep/Wakeup(MC9S12ZVMC), 다이나모미터 토크제어 개발, PCB 6층 설계, 국책과제 완료보고서 |
| **SS500** (핵심) | VCU FW(STM32), CAN 5노드 정합성, 배터리 BMS(48V 460Ah), 상태머신(RC/LCD/ADT), 펌프/팬 전력분석, APQP/DFMEA, 이슈 37건, BOM 132개, 범퍼 안전시스템 |
| **ADT 모터** | 3세대 모터 개선활동, 실주행 데이터 비교, Zbench 지표 60% 향상, TVS Diode 적용 |
| **자동차** (WIA) | G80 BEV 인버터 분석, eLSD 개발 제안, RDC Less 기술 |
| **특허** | 초기위치검출(PN231067KR), 저온기동, 정지감지로직 — 총 3건 |
| **도구** | 오실로스코프, Vector VN1600, ODA EX40-60, SPICE, Matlab/Simulink |

### 실행 지시 — 테마 매핑

아래 후보 테마에 대해 **프로젝트를 넘나드는 공통 업무를 묶어** 각 테마별 STAR 스토리를 구성할 것.

| 테마 후보 | 걸쳐있는 프로젝트 | 핵심 메시지 |
|-----------|------------------|-------------|
| **T-1. 모터제어 깊이** | SCU(BLDC), EOP(SVPWM/저온기동), SS500(CAN 모터제어), ADT(3세대 개선) | "사다리꼴→FOC→SVPWM 진화, 특허 2건, 국책과제" |
| **T-2. CAN 통신 설계** | SCU(CANoe), EOP(CAN Sleep), SS500(5노드 DBC) | "단일 노드→Sleep/Wakeup→다중노드 시스템 진화" |
| **T-3. PCB/HW 설계** | SCU(3세대 PCB), EOP(6층 기판), SS500(VCU REV03) | "설계→시제→양산, 매 세대 개선" |
| **T-4. 시험장비를 만들어서라도 본다** | EOP(다이나모미터), SS500(펌프벤치/팬벤치/범퍼시험) | **아래 Task B 참조** |
| **T-5. 데이터 기반 의사결정** | EOP(온도별 FET 비교), SS500(배터리 SOC, 펌프 전력, 성능비교), ADT(Zbench 60%↑) | "감으로 하지 않고 측정한다" |
| **T-6. 양산 품질 프로세스** | SCU(ES95400/EMC), SS500(APQP/DFMEA/BOM/이슈관리) | "프로토→양산 전환 경험" |
| **T-7. 문제 해결(RCA)** | SS500(동파→오링불량, MR소손→시퀀스재설계, SOC162%→정합성검증) | "현상→원인→대책→검증 루프" |

## Task B. "확인하고 싶으면 장비를 만들어서라도 본다" 스토리

### 목적
포트폴리오에서 가장 차별화되는 엔지니어링 마인드셋 스토리.
"시험 방법이 없으면 만든다" → "데이터로 의사결정한다" → "결과를 정량적으로 보여준다"의 흐름.

### 확인된 시험장비/테스트 기획 사례 (에이전트 결과 종합)

| # | 시험 | 프로젝트 | 만든 것 / 기획한 것 | 정량적 결과 |
|---|------|----------|-------------------|-------------|
| 1 | **다이나모미터 토크제어 시스템** | EOP 400W | Strain Gauge Full Bridge 회로 설계, Wheatstone Bridge 신호처리, 토크 컨트롤러 FW 개발 | 감도 0.996 N·m, 비선형성 0.008%, 10포인트 교정 완료. 사내 세미나까지 개최 |
| 2 | **펌프 성능평가 시스템** | SS500 | Blancett B2800 유량계 + SENSYS PSH 압력계 조합, 테스트베드 단독 구축 | 3개 시료 비교: 10.4/13.0/15.6 LPM, 압력-유량 선형관계 규명 |
| 3 | **팬 시스템 벤치** | SS500 | 스탠드+클램프+풍속계+RS-500X 구성, 거리별(1~5m) 풍속 측정 체계 | Before/After: 갭 45→9mm(80%↓), 출력 19.7→42.2W(110%↑), 감성평가+정량평가 동시 수행 |
| 4 | **범퍼 안전 시험** | SS500 | 속도별 정지거리 6단계 계산 모델 수립 → 실차 검증 | 3km/h: 정지거리 0.082m, 피크전류 308A, 정착시간 467ms |
| 5 | **BLACKTAN 동적 로딩 시험** | SS500 | No-Load/Load 이원화 테스트 설계, 5종 시험(Inrush/정격/과부하/회전/무부하) | 모터 기동~제동 전 구간 특성 데이터 확보 |
| 6 | **PWM 절환 온도 비교** | EOP 400W | SVPWM vs DPWM 비교 시험 설계, 2,932개 데이터 포인트 수집(48분 52초) | FET 온도 16~17°C 저감 실증 |
| 7 | **내구성 시험 기획** | SS500 | 128,000 Cycle Kill Test 설계, Worst 교반 Profile 정의 | 극한조건(온도/습도/지형) 시험 계획 수립 |
| 8 | **ADT 모터 실주행 비교** | ADT | 실주행 데이터 수집 → 개선 전후 비교 분석 | Zbench 2.714→4.346 (60%↑), 리플 139~142ppm 안정화 |
| 9 | **배터리 CAN 정합성 검증** | SS500 | 검증 체크리스트 + 자동 비교 시트 자체 제작 | 3차 반복 검증, 신호별 기대값 vs 실측값 정량 비교 |
| 10 | **SS500 vs 기성모델 비교** | SS500 | 압력/유량/노즐 조건별 비교 시험 설계 | 압력 효율 2배(30bar vs 15bar), 노즐 18 vs 36개 역관계 발견 |
