# 포트폴리오 프로젝트

> Physical AI 시대의 하드웨어 제품 개발자로서의 역량을 어필하는 취업용 포트폴리오.
> 기반 프로젝트: PLUVA FRIEND SS500 (농업용 전동 자율주행 Speed Sprayer)

---

## 포트폴리오 주인공

- **전력제어개발팀 · 선임연구원** — 모터제어 개발자 (본업)
- **제품개발팀 · 주니어 PM** — 제품 기획~양산까지 횡단 역할
- 하나의 제품을 설계부터 양산까지 끌고 가는 **풀스택 하드웨어 엔지니어 + PM**

### 기술 역량

| 영역 | 상세 |
|------|------|
| HW · 전장 | 모터 제어(BLDC/PMSM), 전력 변환, BMS, CAN/CAN-FD |
| Embedded | STM32 FW, RTOS, 상태머신, 센서 인터페이스 |
| 알고리즘 | 제어(PID, FOC), 자율주행 경로 계획, 데이터 기반 튜닝 |
| SW · 코딩 | Python, C/C++, 데이터 분석, CLI 도구, 자동화 |
| 도구 | Git, Notion, CI/CD, CAD 뷰어, 시험 장비 자동화 |
| PM | APQP/PPAP, WBS, 이슈 트래킹, BOM, 일정/원가 관리 |
| AI 활용 | Claude Code 기반 개발 자동화, AI-assisted 워크플로우 |

### 사고 철학

**일론 머스크의 개발 5단계 프로세스**
1. 요구사항을 의심하라
2. 삭제하라
3. 단순화/최적화하라
4. 속도를 높여라
5. 자동화하라
