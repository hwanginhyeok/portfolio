# Case Study: 펌프 성능 업그레이드 — 데이터로 트레이드오프를 보여주다

> 프로젝트: PLUVA FRIEND SS500 (GT-SS500)
> 역할: 전력제어개발팀 선임연구원 (전력 분석 및 업체 서베이 단독 담당)
> 기간: 2025.12 ~ 2026.02
> 이슈 번호: ISSUE #60

---

## Situation

SS500의 분사 시스템 핵심은 **전동 펌프**입니다. 48V 배터리로 구동되는 이 펌프가 과수원에 농약을 분사합니다.

문제는 성능이 **기성 모델(엔진식 100A 펌프)에 비해 부족**하다는 것이었습니다.

| 항목 | SS500 (현재) | 기성 모델 (아세아텍 100A) | 격차 |
|------|-------------|------------------------|------|
| 압력 | 20 Bar | 15 Bar (살포) | SS500 우위 |
| 유량 | 23 L/min | 99 L/min | **4.3배 열위** |
| 노즐 수 | 12개 (1.0mm) | 36개 (1.5mm) | 3배 열위 |

압력은 우위지만, 유량이 4배 이상 부족해서 넓은 과수원에서의 방제 효율이 떨어진다는 현장 피드백이 있었습니다.

---

## Task

**핵심 질문**: "펌프를 업그레이드해야 하는가? 한다면 어디까지?"

단순히 "더 좋은 펌프를 달면 된다"가 아닙니다. 전동 시스템에서 펌프를 키우면:
- 전류가 늘어나고
- 배터리 소모가 빨라지고
- 운전시간이 줄어들고
- 비용이 올라갑니다

이 **트레이드오프를 정량적으로 분석**해서 경영진이 의사결정할 수 있는 데이터를 제공하는 것이 과제였습니다.

---

## Action

### 1단계: 전력 예산 분석

아세아텍의 성능 표를 기준으로, 목표 성능별 필요 전력을 역산했습니다.

**노즐 1.5mm × 19개 (아세아텍 100A 수준)**:
```
펌프 수동력 = P × Q = 1.5MPa × 66LPM = 1.65 kW
모터 입력 = 1.65 / 0.51(효율) = 3.23 kW (4.3마력)
필요 전류 = 3.23kW / 50V = 64.0 A
```

**현재 SS500 (노즐 1.0mm × 12개)**:
```
펌프 수동력 = 2.0MPa × 22.58LPM = 0.75 kW
모터 입력 = 0.75 / 0.55 = 1.36 kW (1.8마력)
필요 전류 = 1.36kW / 50V = 27.0 A
```

**전력 비교 매트릭스**:

| 구성 | 압력 (MPa) | 유량 (LPM) | 수동력 (kW) | 모터입력 (kW) | 전류 (A @50V) |
|------|-----------|-----------|------------|-------------|-------------|
| 현재 (1.0mm×12) | 2.0 | 22.58 | 0.75 | 1.36 | 27.0 |
| 개선안 A (1.0mm×12, 고압) | 3.0 | 17.5 | 0.87 | 1.58 | 31.0 |
| 개선안 B (1.5mm×19) | 1.5 | 66.0 | 1.65 | 3.23 | 64.0 |

### 2단계: 비용 + 운전시간 영향 분석

펌프 업그레이드의 실제 영향을 정량화했습니다:

| 항목 | 현재 (25A급) | 80A급 업그레이드 | 변화 |
|------|------------|----------------|------|
| 펌프 시스템 비용 | 30~40만원 | **~130만원** | **3.3배 증가** |
| 연속 운전시간 | **4시간** | **~2시간** | **50% 감소** |
| 구조 변경 | 없음 | **하부 제외 전면 변경** | 대규모 |
| 모터 급 | 1.8마력 | 4.3마력 | 2.4배 |

### 3단계: 국내외 업체 서베이

**국내 업체 (해청 기계 / 영흥분무기)**:
- SS기 3사(아세아텍, 한아 등)는 100A 모델 적용 — 약 40LPM 수준
- 100A는 7.5HP / 80A는 5.0HP
- 국내 모터 일체형 제품은 중국산이며 성능 미달
- **결론**: 사과 농가 대응을 위해 80A급 필요

**중국 업체 (PLANT MATE)**:
| 항목 | 스펙 |
|------|------|
| 모델 | 26-4z (횡형 4련 플런저식) |
| 크기 | 462 × 420 × 300 mm |
| 무게 | 18 kg |
| 전압 | 48V |
| 유량 | 50~80 L/min |
| 압력 | 40~60 bar |
| 단가 | **1,500위안 (~30만원)** |
| 리드타임 | 20~30대 기준 협의 중 |

→ PLANT MATE의 48V 펌프는 성능(50~80L, 40~60bar)이 목표를 초과하면서 단가(1,500위안)가 현재 시스템 비용 수준이라 유력 후보.

### 4단계: 경영진 의사결정 지원

분석 결과를 하나의 의사결정 매트릭스로 정리해서 보고했습니다:

| 옵션 | 성능 | 비용 | 운전시간 | 구조변경 | 리스크 |
|------|------|------|---------|---------|--------|
| **현상 유지** | 20Bar/23LPM | - | 4h | 없음 | 성능 열위 지속 |
| **PLANT MATE 적용** | 40~60Bar/50~80LPM | +30만원 | ~2h | 중간 | 중국 업체 품질 검증 필요 |
| **국내 80A 적용** | 25Bar/40LPM | +100만원 | ~2h | 대규모 | 비용 부담 |

→ 현재 **보류** 판정. 1차 양산은 현 사양으로 진행하고, PLANT MATE 샘플 평가 후 2차 양산에서 적용 검토.

---

## Result

### 정량적 성과

| 항목 | 성과 |
|------|------|
| 전력 분석 | 3개 구성별 압력/유량/수동력/모터입력/전류 완전 산출 |
| 비용 분석 | 현재 대비 3.3배(130만원) 증가 정량화 |
| 운전시간 영향 | 4h → 2h (50% 감소) 정량화 |
| 업체 서베이 | 국내 2사 + 중국 1사 컨택, 스펙/단가/리드타임 확보 |
| 의사결정 지원 | 3개 옵션 매트릭스로 경영진 판단 근거 제공 |

### 비즈니스 임팩트

1. **양산 일정 보호**: 성능 욕심에 양산이 밀리는 것을 방지. "1차 양산은 현 사양, 2차에서 개선"이라는 합리적 로드맵 수립
2. **비용 투명화**: "좋은 펌프 달자"라는 모호한 요구를 "130만원 + 구조 전면 변경 + 운전시간 반감"으로 구체화
3. **대안 확보**: PLANT MATE 발굴로 성능은 올리면서 비용은 현 수준 유지 가능한 옵션 제시

## 면접 예상 Q&A

**Q: 결국 업그레이드를 안 한 건가요? 실패 아닌가요?**

> "보류"는 실패가 아니라 **데이터 기반의 의사결정**입니다. 업그레이드하면 비용 3배 + 운전시간 반감이라는 사실을 모른 채 진행했다면, 그게 더 큰 실패입니다. 1차 양산 16대의 출하 일정을 지키면서, PLANT MATE 샘플 평가를 병행하는 것이 리스크와 기회의 균형점이었습니다.
